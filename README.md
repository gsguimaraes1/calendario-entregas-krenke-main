# Monitor de Entregas Krenke

Monitor operacional em tempo real desenvolvido para visualização em TV, integrando dados de entregas via CSV.

## Funcionalidades
- **Drill-Up/Down**: Visualize por Dia, Semana, Mês ou Ano.
- **Auto-Refresh**: Atualização automática dos dados a cada 30 minutos.
- **Modo TV**: Layout de alto contraste e fontes grandes para leitura à distância.
- **Sincronização**: Sincronizado diretamente com a planilha de entregas.

## Tecnologias
- React + TypeScript
- Vite
- Tailwind CSS
- TanStack Query
- Lucide Icons

## Execução Local
```bash
npm install
npm run dev
```
